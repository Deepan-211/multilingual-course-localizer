# multilingual-course-localizer

AI-Powered Multilingual Content Localization Engine for Skill Courses

## Overview

This project localizes skill-based course content into multiple languages using AI. It helps learners access educational content in their preferred language while preserving meaning, context, and learning objectives.

## Tech Stack

### Frontend

* Next.js 14+ (App Router)
* TypeScript
* Tailwind CSS
* shadcn/ui

### Backend

* Spring Boot
* Java

### Database

* MySQL

### AI Integration

* Claude API / LLM

## Features

* Course upload and management
* AI-powered content translation
* Multi-language support
* Localization progress tracking
* User authentication
* Course workspace dashboard
